==================================
FastPt XML Templates
==================================

Tags
====

- Syntax-related tags (these don't expand)
  - py:content
  - py:replace
  - py:attrs
  - py:strip
  - py:nop
- Control-flow
  - py:with
  - py:switch
  - py:case
  - py:else
  - py:for
  - py:if
- Module-related tags
   - py:extends
   - py:import
   - py:include
- Function-related tags
   - py:def
   - py:block
   - py:call
   - py:super (?) -- ${parent_block()}
